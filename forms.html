Form elements

Forms

Text, phone, select, checkbox, and switch — glass-styled inputs that pick up whatever theme is active.

Text input

const field = rg.ui.input({
  label: 'Name',
  placeholder: 'Ada Lovelace',
  onInput: (value) => console.log(value)
});
document.body.appendChild(field.element);

Phone input

Formats digits as you type — (555) 123-4567 — and hands back both the formatted string and the raw digits.

const phone = rg.ui.phoneInput({
  label: 'Phone number',
  onChange: (formatted, digits) => console.log(formatted, digits)
});
document.body.appendChild(phone.element);

Select

const field = rg.ui.select({
  label: 'Orbit',
  options: [
    { value: 'leo', label: 'Low Earth Orbit' },
    { value: 'meo', label: 'Medium Earth Orbit' },
    { value: 'geo', label: 'Geostationary' }
  ],
  value: 'leo',
  onChange: (value) => console.log(value)
});

Checkbox

rg.ui.checkbox({
  label: 'Notify me on deploy',
  checked: true,
  onChange: (checked) => console.log(checked)
});

Switch

const toggle = rg.ui.switchToggle({
  label: 'Auto-refresh',
  onChange: (checked) => console.log(checked)
});
document.body.appendChild(toggle.element);

Debounced search

Waiting for input…

const search = rg.utils.debounce((value) => {
  console.log('Searching for:', value);
}, 400);

field.input.addEventListener('input', (e) => search(e.target.value));
