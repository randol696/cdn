rg cdn

A personal library of UI elements, styles, and utility functions — glass-panelled, dark or light, served straight from GitHub Pages.

Quick start

Drop these two tags into any page. No build step, no dependencies.

<link rel="stylesheet" href="https://randol696.github.io/cdn/css/rg.css">
<script src="https://randol696.github.io/cdn/js/rg-lib.js"></script>

Prefer a versioned, edge-cached copy? Point at jsDelivr's GitHub mirror instead — swap @main for a tag once you start cutting releases: 

<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/randol696/cdn@main/css/rg.css">
<script src="https://cdn.jsdelivr.net/gh/randol696/cdn@main/js/rg-lib.js"></script>

Everything hangs off one global, rg: rg.ui.* for elements, rg.utils.* for helpers. Explore Components, Forms, and the full Reference in the nav above.

Theming

Every color is a CSS variable, so the whole kit — including this site — flips between dark and light with one call. Try the toggle in the top-right corner.

rg.utils.theme.init();      // apply saved preference on page load — call once, early
rg.utils.theme.toggle();    // dark <-> light
rg.utils.theme.set('light');
rg.utils.theme.get();       // 'dark' | 'light'

Skins

Dark/light controls light vs. dark; skins control the accent color pair on top of that — five built in, and independent of theme. Try the swatches next to the theme toggle.

rg.utils.skin.init();          // apply saved preference on page load — call once, early
rg.utils.skin.set('violet');   // 'cyan' | 'violet' | 'emerald' | 'amber' | 'rose'
rg.utils.skin.get();

document.body.appendChild(rg.ui.skinPicker()); // ready-made swatch row

Buttons

Eight color variants, and shape is opt-in — pill by default, or rounded / square.

rg.ui.button({ text: 'Primary', variant: 'primary' });
rg.ui.button({ text: 'Success', variant: 'success' });
rg.ui.button({ text: 'Warning', variant: 'warning' });
rg.ui.button({ text: 'Info',    variant: 'info' });
rg.ui.button({ text: 'Danger',  variant: 'danger' });
rg.ui.button({ text: 'Outline', variant: 'outline' });
rg.ui.button({ text: 'Ghost',   variant: 'ghost' });

// shape: 'pill' (default) | 'rounded' | 'square'
rg.ui.button({ text: 'Square', shape: 'square' });

Card & badge

rg.ui.card({
  title: 'Deploy status',
  body: 'Last build finished without errors.',
  footer: 'Updated just now'
});

rg.ui.badge('online', 'success');

Alerts

rg.ui.alert({
  title: 'Heads up',
  body: 'This banner stays until dismissed.',
  variant: 'warning'   // default | success | warning | danger
});
