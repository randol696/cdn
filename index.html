RG CDN

A personal library of UI elements, styles, and utility functions — served straight from GitHub Pages.

Quick start

Drop these two tags into any page. No build step, no dependencies.

<link rel="stylesheet" href="https://randol696.github.io/cdn/css/rg.css">
<script src="https://randol696.github.io/cdn/js/rg-lib.js"></script>

Prefer a versioned, edge-cached copy? Point at jsDelivr's GitHub mirror instead — swap @main for a tag once you start cutting releases: 

<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/randol696/cdn@main/css/rg.css">
<script src="https://cdn.jsdelivr.net/gh/randol696/cdn@main/js/rg-lib.js"></script>

Everything hangs off one global, RG: RG.ui.* for elements, RG.utils.* for helpers.

Buttons

RG.ui.button({ text: 'Primary', variant: 'primary' });
RG.ui.button({ text: 'Secondary', variant: 'secondary' });
RG.ui.button({ text: 'Danger', variant: 'danger' });
RG.ui.button({ text: 'Outline', variant: 'outline' });

Card & badge

RG.ui.card({
  title: 'Deploy status',
  body: 'Last build finished without errors.',
  footer: 'Updated just now'
});

RG.ui.badge('online', 'success');

Toast

RG.ui.toast('Saved!', { variant: 'success', duration: 2500 });

Modal

RG.ui.modal({
  title: 'Confirm',
  body: 'Are you sure you want to continue?'
});

Spinner

RG.ui.spinner(32);

Utilities

Debounced search

Waiting for input…

const search = RG.utils.debounce((value) => {
  console.log('Searching for:', value);
}, 400);

input.addEventListener('input', (e) => search(e.target.value));

Copy this page's URL to clipboard

RG.utils.copyToClipboard(window.location.href);
